Tubeness

Tubeness (ImageJ)
Author Mark Longair, Stephan Preibisch
Maintainer Mark Longair
File VIB_.jar
Source on GitHub
Development status stable




A multithreaded implementation of the original IJ1 Tubeness plugin has been developed by Jean-Yves Tinevez in the context of ImageJ_Ops.


The original documentation for the IJ1 Tubeness plugin can be found at http://homepages.inf.ed.ac.uk/s9808248/imagej/tubeness/.
