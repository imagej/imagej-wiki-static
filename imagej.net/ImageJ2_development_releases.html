ImageJ2 development releases

This table lists the versions of ImageJ2 released during its initial incubation period.

Version Code name Release date Theme Roadmap

2.0.0-alpha4 - 2011 August 1 Various features and bugfixes imagej-2.0-alpha4
2.0.0-alpha5 - 2011 September 13 Robust ImageJ1 compatibilty imagej-2.0-alpha5
