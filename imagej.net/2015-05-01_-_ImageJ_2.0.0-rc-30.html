2015-05-01 - ImageJ 2.0.0-rc-30

Today, the ImageJ team has made a new release of ImageJ2: version 2.0.0-rc-30. This release includes bug fixes and usability improvements, especially for scripting.

In addition, the Fiji team has updated several components of the Fiji distribution.

What's new

These releases include new versions of several components:

SciJava ImageJ Fiji
  • scijava-ui-swing 0.6.1
  • scripting-jython 0.2.1
  • scijava-common 2.41.0
  • imagej-legacy 0.15.0
  • imagej-ops 0.12.0
  • imagej-ui-swing 0.11.2
  • imagej-updater 0.7.4
  • imagej-plugins-commands 0.5.1
  • SPIM_Registration 2.3.2
  • bigdataviewer-core 1.1.3
  • Trainable_Segmentation 2.2.0

Highlights

  • ImageJ2 commands and scripts are now macro-recordable (imagej-legacy@02e9e462).
  • Script parameters now have reasonable default values (scijava-ui-swing@7dc0af5e) which can be explicitly set (scijava-common@3fdc5df4).

How to update

Use the Help  › Update... command to update your ImageJ installation. Please send any comments to the ImageJ mailing list. Thank you for the continued feedback and support!
