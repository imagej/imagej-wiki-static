When to use

Put it on top of a page describing a plugin that has been removed from the latest Fiji distribution. Using the {{{reason}}} field, put a justification for the removal.

Example

{{PluginRemoved 
| reason = It was not good enough, and another new plugin does what this one did in a better way.
| date = 28 April 2009 
}}
