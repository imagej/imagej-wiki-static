Category:Macro Tools

Collection of pages that describe macro tools appearing in the ImageJ Menu, as defined in the ImageJ documentation.

There are 3 types of Macro Tools:

- Image Tools : executes when the user clicks on the image with that tool

- Action Tools : perform an action when you click on their icon in the toolbar (ex: About ImageJ)

- Menu Tools : Add new drop-down menu to the ImageJ toolbar
