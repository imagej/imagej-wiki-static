N5

N5 is a library and API for storing large chunked n-dimensional tensors, and arbitrary meta-data in a hierarchy of groups similar to HDF5.

The N5 update site distributes the N5 libraries so they can be used in ImageJ scripts.

See the N5 repository for further details.
