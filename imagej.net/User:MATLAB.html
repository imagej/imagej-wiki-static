User:MATLAB

This account is used to manage the MATLAB update site. See also list of update sites.

To activate it:

  • Help>Update
  • Click Manage update sites
  • Check MATLAB
  • Click Close

Some files should appear on the list with Status/Action of Install it.

  • Click Apply changes
  • Restart ImageJ

Your Script Editor should now have MATLAB as an available language option!
