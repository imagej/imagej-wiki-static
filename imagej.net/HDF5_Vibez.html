HDF5 Vibez

This plugin is an HDF5 reader/writer plugin for ImageJ and Fiji. It is a continuation of the original code by Matthias Schlachter, Olaf Ronneberger and Kun Liu.

HDF5 is a data format for storing extremely large and complex data collections. For more information, see the official website. The plugin uses the jhdf5 library from ETH SIS for reading and writing HDF5 files.

For further details, see the fiji/HDF5_Vibez repository on GitHub.
