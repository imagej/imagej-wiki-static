Widget:UsageMap

This widget allows embedding of ImageJ usage maps on wiki pages.

Using this widget

Supported parameters include:

  • url - HTML page of the map to embed
  • border - frameborder width
  • width - width of the map (default is 400)
  • height - height of the map (default is 300)

Copy to your site

To use this widget on your site, just install the MediaWiki Widgets extension and the copy full source code of this page to your wiki, as an article called Widget:UsageMap.
