Recommended ImageJ Plugins for Tracking

Here we summarize some of the tracking plugins in ImageJ.

Name Short Description Highlights Plugin Snapshot
TrackMate A tool for automated, and semi-automated particle tracking.
  • Versatile and modular solution that works out of the box for end users
  • Provides several visualization and analysis tools that aid in assessing results
  • A very modular design, that makes it easy to extend
