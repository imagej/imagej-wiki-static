2009-07-08 - Synchronized with ImageJ 1.43b

We synchronized our version of ImageJA with the upstream version ImageJ 1.43b. This finally allows compiling our version with OpenJDK, as ImageIO rather than Sun's proprietary API is used to read and write Jpeg images.
