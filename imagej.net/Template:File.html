Template:File


Use this template to show a file box with some content.

Parameters

1. unnamed
Add the filename.
2. unnamed (optional)
Add some description (optional as can be blank).
3. unnamed
Add the code/file content.

Usage

The following: 

{{File|filename||<pre>
file content
  this
    that
</pre>}}

generates:

Filefilename

file content
  this
    that
