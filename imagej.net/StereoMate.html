StereoMate

StereoMate (FIJI)
Author Steven J. West
Maintainer Steven J. West
File Stereo_Mate.jar
Source [1]
Initial release August 2019
Development status stable
Website [2]


Purpose

This plugin suite provides tools for the processing and analysis of confocal image stacks. It aims to allow the stereological and automated analysis of 3D reconstructed objects from regions of interest.

Documentation

This plugin provides the image processing and analysis functionality described in the following paper West, S-J. & Bennett, D-H-L. (August 2019), "StereoMate: A Stereological and Automated Analysis Platform for Assessing Histological Labelling in Cleared Tissue Specimens", bioRxiv, doi:https://doi.org/10.1101/648337, <https://www.biorxiv.org/content/10.1101/648337v2>
