Dynamic ROI Profiler

Dynamic ROI Profiler (Clojure)
Author Albert Cardona
Maintainer
File Dynamic_ROI_Profiler.clj
Source on GitHub
Latest version 16 November 2008
Development status unknown



Dynamically updates a profile plot for an image with a line, freeline, polyline or a rectangular ROI, as the ROI is moved across the image. Written in Clojure.

Dragging a line ROI across an image shows the pixel intensity profile across that line in a separate image window.
