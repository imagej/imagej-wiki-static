ImageJFX

ImageJFX (ImageJ)
Author Cyril Mongis
Maintainer Cyril Mongis
Source on GitHub
Development status dormant
Website http://www.imagejfx.net/

The ImageJFX project aims to create a new user interface for ImageJ in order to ease scientific image analysis. While keeping the core components of ImageJ, ImageJFX brings scientists closer to their goal by making the interface clearer for beginners and more practical for advanced users.

For further details, see the ImageJFX web site.
