User:LOCI

This account is used to manage the LOCI update site. See also list of update sites.

To activate it:

  • Help › Update...
  • Click Manage update sites
  • Check LOCI
  • Click Close

Some files should appear on the list with Status/Action of Install it.

  • Click Apply changes
  • Restart ImageJ

Your ImageJ should now possess additional commands in the Plugins › LOCI menu.
