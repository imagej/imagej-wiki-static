2010-06-03 - New SPIM registration plugin - released and published in Nature Methods

After years of development we release today through the Fiji updater a new plugin for registration of multi-view Selective Plane Illumination Microscopy (SPIM) datasets. Besides the exciting applications for SPIM imaging the code base can be used for efficient matching of large, partially overlapping 3d point clouds. The software was published in Nature Methods. Details on how to use the plugin for SPIM registration can be found at SPIM Registration and the principle of the algorithm is extensively documented at SPIM Registration Method. Feedback is very welcome on this brand new Fiji project.
